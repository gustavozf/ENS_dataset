ENS Dataset

==========================================
* Number of image samples (per type): 416

* 3 Versions for each sample:
	- Enteric Glial Cells  (EGC)
	- Enteric Neuron Cells (ENC)
	- Merged = EGC + ENC (MRG)

* Image Dimensions: 1384 x 1036
==========================================
Image samples' name pattern:

	TAG_ID_TYPE.png

* TAG: Animal's tag. Composed of a number and the animal's class.

       Being:
       - C: control (or healthy)
       - TW: sick (w/ cancer)

* ID: Sample's identifier number. 
      In total, there are 32 image samples per animal.

* TYPE: Image sample version.
	(EGC, ENC or MRG)

Example:
	3C_10_ENC.png
